## 📁 시작 가이드

### Requirements

#### 🖥️ Backend
- Python 3.10 이상  
- Django 4.0  
- Django REST Framework  
- PostgreSQL  
- Kakao OCR API

#### 💻 Frontend
- Node.js 14.19.3  
- Npm 9.2.0  
- React.js  
- Axios  
- Chart.js

#### 🎨 Design
- Figma  
- Notion (플로우 차트 및 명세 관리용)

# 학과 통계 열람기

---

## 프로젝트 소개

성적표 한 장으로 졸업 요건, 이수 현황, 누락 과목까지 자동으로 정리해주는 웹 대시보드 서비스입니다.

---

## 주요 기능

📸 성적표 이미지 업로드 (캡처 기준)

🔍 OCR로 교과목명, 학점, 학기 추출

📊 전공/교양 이수율 계산 + 졸업요건 자동 체크

🎯 현재는 25학번 컴퓨터공학과만 지원, 이후 확장 예정
